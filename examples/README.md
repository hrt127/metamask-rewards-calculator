# Examples Directory

This directory contains additional resources and content derived from the MetaMask Rewards Calculator analysis.

## Files

### `telegram-message.md`
A condensed version of the analysis formatted for sharing in Telegram or other messaging platforms. Includes:
- Core numbers and tables
- Quick decision framework
- Key risk factors
- Links to resources

Perfect for copy-pasting into group chats or DM threads.

### `detailed-visual-analysis.md`
An expanded deep-dive version of the analysis. Includes:
- Extended executive summary
- Level-by-level breakdown with verdicts
- Two conversion scenarios explained
- Route comparison with efficiency ratings
- Optimization strategy walkthrough
- Decision tree framework
- Risk assessment

Use this for thorough research or longer-form content sharing.

## Usage

These files are **derived from the calculator** and represent different presentations of the same underlying analysis. They are:

- ✅ Read-only reference materials
- ✅ Educational content
- ✅ Community sharing resources

For the actual calculator and to run your own analyses, use the main `calculator.js` in the root directory.
